# ドローン飛行範囲作成マップ

社内のドローン飛行申請用に、スマホやPCのブラウザで国土地理院地図へ「飛行予定範囲」と「離陸場所」を描き、PDFとして保存するための静的Webアプリです。

サーバーやデータベースは使わず、HTML / CSS / JavaScriptのみで動作します。GitHub Pagesでそのまま公開できます。

## 使い方

1. `index.html` をブラウザで開きます。
2. 必要に応じて「現在地」ボタンで現在地周辺へ移動します。
3. 「離陸場所を指定」を押し、地図上をタップして離陸場所を登録します。
4. 「飛行範囲を描く」を押し、地図上を順番にタップして飛行予定範囲を囲みます。
5. 地図左上の編集・削除ボタンで、描いた飛行範囲を修正できます。
6. 申請情報欄に、取材名、撮影予定日、場所、操縦者、補助者、備考を入力します。
7. 「PDF保存」を押し、ブラウザの印刷画面からPDFとして保存します。
8. 入力内容と図形を消す場合は「全消去」を押します。

## GitHub Pagesでの公開方法

1. このフォルダのファイル一式をGitHubリポジトリへアップロードします。
   - `index.html`
   - `style.css`
   - `app.js`
   - `README.md`
2. GitHubのリポジトリ画面で `Settings` を開きます。
3. `Pages` を開きます。
4. `Build and deployment` の `Source` で `Deploy from a branch` を選びます。
5. 公開したいブランチを選び、フォルダは `/root` を選択します。
6. 保存後、表示されたGitHub PagesのURLへアクセスします。

## 注意事項

このアプリは社内申請用の飛行範囲図を作るための補助ツールであり、法規制の自動判定は行いません。

DID、空港周辺、緊急用務空域などの法規確認は、SORA map等の公式情報で別途確認すること。

DID、空港周辺、緊急用務空域、150m以上、第三者上空などの確認は、必ず国土交通省の公式情報やSORA map等で別途確認してください。
